# WeatherVision (GitHub Pages)

Профессионально оформленный погодный сервис для **телефона / планшета / ПК**:

- Поиск города
- Геолокация
- Текущая погода + «ощущается»
- Почасовой прогноз (24 часа)
- Прогноз на 7 дней
- Карта (Leaflet)
- Избранные города (localStorage)
- Тема: auto / light / dark

## Данные
- Weather + AQI: **Open‑Meteo**
- Карта: **OpenStreetMap + Leaflet**

## Запуск локально
Открой `index.html` в браузере.

## Деплой на GitHub Pages
Положи файлы в ветку `main` (или `gh-pages`) и включи Pages.
